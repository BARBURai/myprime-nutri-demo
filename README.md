# MyPrime — דמו תזונה (v0.11)

אפליקציית ווב (Vite + React) לדמו של מעקב תזונה. גרסה אחת חיה, מנוהלת ב-Git, מוכנה לפריסה.

## מה עובד ????
- מסכי אונבורדינג, בית, יומן, דוח, מתכונים, פרופיל
- מנוע גרעון/מאקרו, יעד מים, יעד חלבון, התקדמות לפי שבוע
- **בדף מתארח (HTTPS):** צילום צלחת ודיבור עובדים (מצלמה/מיקרופון של הדפדפן)
- **ברקוד** — הדגמה בשלב זה
- **ניתוח תמונה/דיבור ע"י AI** — עובד רק כשמוגדרת פונקציית שרת + מפתח (ראו למטה). אם לא — נופל בחן לערכי דוגמה.

## הרצה מקומית (אופציונלי, דורש Node)
```
npm install
npm run dev
```
ואז פותחים את הכתובת שמודפסת (בדרך כלל http://localhost:5173).

## העלאה ל-Git ופריסה — המסלול הקל (בלי טרמינל)
1. צרו ריפו חדש ב-GitHub (github.com → New repository).
2. גררו את כל הקבצים מהתיקייה הזו אל הריפו (Add file → Upload files) ושמרו (Commit).
3. היכנסו ל-vercel.com → New Project → Import את הריפו → Deploy. Vercel מזהה Vite אוטומטית.
4. מקבלים כתובת חיה (למשל myprime-demo.vercel.app). זו הכתובת שתשלחו לענת — תמיד מציגה את האחרון.

## העלאה ל-Git — מסלול טרמינל
```
git init
git add .
git commit -m "MyPrime demo v0.11"
git branch -M main
git remote add origin <כתובת-הריפו-שלך>
git push -u origin main
```

## הפעלת ניתוח ה-AI (צילום/דיבור)
המפתח אסור שייחשף בדפדפן, לכן הקריאות עוברות דרך `api/ai.js` (פונקציית שרת ב-Vercel).
ב-Vercel: Project → Settings → Environment Variables, והוסיפו:
- `ANTHROPIC_API_KEY` = המפתח שלכם
- (אופציונלי) `AI_MODEL` = מזהה מודל Sonnet עדכני

בלי המפתח — האפליקציה עובדת רגיל, ופשוט מציגה ערכי דוגמה בניתוח.
